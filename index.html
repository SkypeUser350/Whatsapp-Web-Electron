WhatsApp Web Electron

An isolated desktop client experience powered by NW.js, delivering a clean web interface container without native resource drains.

View GitHub Repository
Why use this client? It provides a standalone desktop environment for WhatsApp Web that is entirely free from WebView2 reliance, making it a fast and clean option for your desktop.

How to Install

  • Step 1

    Get NW.js

    Go to nwjs.io and download the Stable package with the Normal option.

  • Step 2

    Download Repository Files

    Go to this repository, click on the green Code button, and select Download ZIP to extract the files.

  • Step 3

    Assemble and Launch

    Extract your downloaded NW.js folder, then copy and paste the package.json from this repository directly inside it. Run nw.exe to open WhatsApp Web!

Customization & Shortcuts

  • Optional

    Add Application Icon

    Download the Whatsapp.ico file from this repository and set it as the icon for your newly configured application folder.

  • Important

    Pin to Start Menu or Taskbar

    To easily access your app later, open nw.exe, right-click its taskbar icon, or drag a shortcut of nw.exe onto your desktop/Start Menu.
