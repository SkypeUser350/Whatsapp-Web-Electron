WhatsApp Web Electron

A seamless, lightweight Electron wrapper for WhatsApp Web. Enjoy a dedicated desktop chat experience without the browser clutter.

Download Latest Release View on GitHub

Why Choose This Client?

🚀 Lightweight

Stripped of unnecessary browser bloat, focus solely on your messages with minimal resource consumption.

🔔 Native Feel

Runs in its own independent window, making it easy to manage via your OS taskbar or dock.

🔒 Secure

Loads official WhatsApp Web code directly. No middleman, keeping your personal data safe and encrypted.

Quick Build Setup

Want to run it from source? Clone and spin it up in seconds:

# Clone the repository
git clone https://github.com

# Navigate into the directory
cd Whatsapp-Web-Electron

# Install dependencies
npm install

# Start the application
npm start
