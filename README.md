# Whatsapp-Web-Electron
This is Whatsapp Web but electron and its actually better than webview2

# How to install?

# Step 1 

get nwjs from nwjs.io and pick stable and normal option

# Step 2

download it and also download the package.json from this repo

# Step 3 

After nwjs is downloaded and extracted put package.json in it

# And Now You Got Whatsapp Web electron free from webview2 

# OPTIONAL

if u want to also add an whatsapp icon download the whatsapp.ico from this repo and then put it as your icon for app

# IMPORTANT 

if u want to pin it to start menu or taskbar or desktop 

Step 1 : open dw.exe and then right click on it 
Step 2 : and drag it to desktop to add icon or pin it to start menu or taskbar
